public function callbackEvent(target:Object, chanel:String = null, data:Object = null):void

Вызвать метод обратного вызова. Метод callbackEvent вызывается у всех зарегистрированных(родительских) объектов.

Parameters

chanel:String (default = null) — Канал обратного вызова, значение по умолчанию FPCaller.DEFAULT="default"
