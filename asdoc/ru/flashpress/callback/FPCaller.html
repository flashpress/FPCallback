callbackEvent()method 
public function callbackEvent(target:Object, chanel:String = null, data:Object = null):void

Вызвать метод обратного вызова. Метод callbackEvent вызывается у всех зарегистрированных(родительских) объектов. Метод также запускает функцию parentCallEvent, если она была задана в конструкторе.

Parameters

target:Object — Инициатор обратного вызова
 
chanel:String (default = null) — Канал обратного вызова, значение по умолчанию FPCaller.DEFAULT="default"
 
data:Object (default = null) — Информация передаваемая при обратном вызове
